

# MEAN stack basics

 M = MongoDB
 E = Express Webserver
 A = angular.js
 N = 

The A in MEAN stands for Angular js on the client side
This runs in a browser using MVC
  angular.js
     Model
       JSON/XML/JavasScript objects
     View
       HTML/CSS
     Controller
       Angular.js scripts
     User
       interactions      

## Usage

  View --> (html + css) --> HTTP GET/PUT/AJAX --> (json + jpg + js)  --> node.js (express Webserver + server side scripts)
